Welcome to Chess Saver!

The One-Stop-Shop for all your chess position saving needs!

USERS:

  • Head to Load to load a position FROM the server
  • Head to Save to save a position TO the server

DEVELOPERS:

These endpoints will get a position from the server either randomly, by name, or simply all of them!!

  • /random-position
  • /random-position?name=start
  • /all-positions

If an invalid name is given as a parameter, the API will simply return a random position instead

ADMINS:

  • Head to Admin to see what positions are saved on the server!
  • Check out the /404Page (if you haven't already due to erroneous typing)
